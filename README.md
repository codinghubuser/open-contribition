# open-contribition
<h3>The is repository is used for practicing git and github concepts</h3>
<h4>Steps</h4>
1.Fork this repository<br/>
2.clone this repo to your local computer using git clone "url" command<br/>
3.create a new branch with your username as branch name<br/>
4.Add your github username to username.md file<br/>
5.commit those changes <br/>
6.push it to origin<br/>
7.create a pull request <br/>

By following above steps you sucessfully learned how to use git and github for realworld projects<br/>
